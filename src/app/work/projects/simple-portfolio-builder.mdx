## Overview

TeleShop Bot is a Telegram-based e-commerce assistant that facilitates crypto payments and order automation in chat. It simplifies purchasing workflows for customers by combining messaging, transactions, and product management in one bot.

## Key Features

- **Crypto Payments**: Allows customers to complete purchases using cryptocurrency directly within Telegram.
- **Chat Automation**: Automates product discovery, checkout and status updates.
- **Secure Transactions**: Uses secure backend verification for payment processing.
- **User-friendly Experience**: Designed for smooth engagement with minimal friction in chat.

## Technologies Used

- **Telegram Bot API** for interactive messaging flows.
- **Python** for backend automation and payment orchestration.
- **Blockchain payment integration** for crypto checkout.
- **Docker** for deployment and environment consistency.

## Outcome

TeleShop Bot demonstrates a practical bridge between decentralized payments and conversational commerce, making crypto transactions more accessible and user-friendly.
